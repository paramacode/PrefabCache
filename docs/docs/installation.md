# Installation

You’ve got three options to install **PrefabCache** into your *project:*

### *NPM*

Intended only for *[roblox-ts](https://roblox-ts.com/)* users:
Install **PrefabCache** by running *the following command* in your *terminal*:

```bash
npm install @rbxts/prefabcache@latest
```

### *WALLY*

Install **PrefabCache** using a popular package manager called *[Wally](https://wally.run/)*. Put the following line
under `[dependencies]` in your `wally.toml` file.
*Consider using the latest version*:

```md title="wally.toml"
[dependencies]
PrefabCache = "paramacode/prefabcache@1.0.0"
```

### *ROBLOX STUDIO*

If you are not using **[Rojo](https://rojo.space/)**, navigate yourself to *[github releases](https://github.com/paramacode/PrefabCache/releases/)*
page and download *rbxm* file of *the latest* version.
